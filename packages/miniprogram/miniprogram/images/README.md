# 图标说明

## TabBar 图标

需要准备以下图标文件（建议尺寸 81x81px，PNG 格式）：

| 文件名 | 用途 | 建议图标 |
|--------|------|----------|
| tab_home.png | 首页（未选中） | 🏠 灰色房子 |
| tab_home_active.png | 首页（选中） | 🏠 粉色房子 |
| tab_heart.png | 收藏（未选中） | 🤍 灰色爱心 |
| tab_heart_active.png | 收藏（选中） | ❤️ 粉色爱心 |
| tab_settings.png | 设置（未选中） | ⚙️ 灰色齿轮 |
| tab_settings_active.png | 设置（选中） | ⚙️ 粉色齿轮 |

## 分享图

| 文件名 | 用途 | 建议尺寸 |
|--------|------|----------|
| share.png | 分享卡片封面 | 500x400px |

## 颜色规范

- 主色（选中态）: #FF6B8A
- 灰色（未选中）: #999999
- 背景色: #FFFFFF

## 快速生成

可以使用在线工具生成：
1. https://www.iconfont.cn/
2. https://iconpark.oceanengine.com/

搜索 "home" "heart" "settings" 下载对应图标即可。

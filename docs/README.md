Welcome to the documentation folder for Orbit.

## File Structure

- **`original_concept_zh.md`**: The original product thought process, viral mechanics, and MVP definition provided in the initial draft.
- **`orbit_product_logic.md`**: The upgraded product logic for Orbit. This includes the updated "Bento-box" styling notes, Web3 credential integrations, and the vision for the Chrome Extension.

## The Orbit Vision

Orbit is designed to be an omnipresent, lightweight Web3 identity layer. Rather than forcing users into a heavy standalone application, Orbit meets them where they already hang out. 

**Form Factors:**
1. **Viral Mini-App / PWA**: Shared via deep links on Telegram, Discord, and Line.
2. **Chrome Extension (Future Phase)**: An integrated browser experience that reads handles/wallets on legacy Web2 platforms (like X/Twitter) and seamlessly overlays the user's Orbit identity card.
